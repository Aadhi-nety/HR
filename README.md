# HR Workflow Designer

A modern and visual workflow designer built with **React** and **React Flow** for creating and testing HR workflows such as onboarding, leave approval, and document verification.

## üéÆ How to Use
### Building a Workflow
- Drag nodes from the palette onto the canvas
- Connect nodes by dragging from one node's handle to another
- Click on nodes to configure their properties
- Use controls to zoom, pan, or fit the view

---

### Testing Workflows
- Click "Test Workflow" in the sandbox panel
- View step-by-step execution in the log
- Check validation results for errors
- See workflow statistics for insights
